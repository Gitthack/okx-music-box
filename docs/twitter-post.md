如何把市场数据变成音乐？

🎵 价格波动 → 音高
+10% → 🎺 高音Do (C6)
+5%  → 🎸 中音Mi (E5)  
0%   → 静音
-5%  → 🎹 低音La (A3)

⚡ 交易量 → 节奏
高交易量 → 快节奏 (120 BPM)
低交易量 → 慢节奏 (60 BPM)

💎 市值 → 音色
大市值 → 🎹 钢琴 (稳重)
小市值 → 🎹 电子合成器 (激进)
