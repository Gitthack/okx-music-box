# 🎵 DeFi Music Box

> 市场波动生成音乐 - 让你的DeFi交易变成旋律

## 💡 核心概念

**"涨跌皆有声，交易即音乐"**

DeFi Music Box 是一个独特的**音频可视化DeFi仪表盘**。它将枯燥的数字变成悦耳的音乐：
- 📈 价格上涨 → 高音阶
- 📉 价格下跌 → 低音阶  
- ⚡ 交易量 → 节奏快慢
- 💧 流动性 → 混响效果

**不是看K线，是听K线！**

## 🎼 音乐映射规则

```
价格波动 → 音高
├─ +10%   → 高音Do (C6)
├─ +5%    → 中音Mi (E5)
├─ 0%     → 静音
├─ -5%    → 低音La (A3)
└─ -10%   → 超低音Do (C2)

交易量 → 节奏
├─ 高交易量 → 快节奏 (120 BPM)
└─ 低交易量 → 慢节奏 (60 BPM)

市值 → 音色
├─ 大市值 → 钢琴 (稳重)
└─ 小市值 → 电子合成器 (激进)
```

## 🎮 核心功能

### 1. 实时市场交响乐 🎺
选择代币组合，生成实时市场音乐：
```
BTC ($67,000) → 🎹 钢琴低音
ETH ($3,450)  → 🎸 吉他中音  
PEPE (+15%)   → 🎺 小号高音
```

### 2. 交易旋律回放 🎧
回放历史交易，用音乐讲述你的交易故事：
```
"2024-03-08 14:32"
买入ETH 0.5个
价格: $3,200 → $3,450 (+7.8%)
🎵 旋律: Do-Mi-Sol (上升三和弦)
情绪: 😊 欢快胜利曲
```

### 3. AI作曲师 🤖
AI分析市场趋势，创作独家"市场交响曲"：
```
🤖 AI: "检测到 bullish 趋势"
🎼 作曲: 《牛市狂想曲》
风格: 激昂、快速、大调
时长: 30秒
可导出为 NFT 音乐作品
```

### 4. 交易DJ台 🎛️
成为DJ，用你的交易打碟：
```
左声道: ETH价格
右声道: BTC价格
混音: 你的交易节奏
效果: Gas费波动 → 滤波器
```

## 🛠️ 技术栈

```
DeFi Music Box
├── 数据层 (OKX Skills)

│   ├── okx-dex-token     → 代币列表
│   └── okx-wallet-portfolio → 个人交易历史
├── 音乐引擎
│   ├── Web Audio API     → 实时合成
│   ├── Tone.js           → 音乐逻辑
│   └── 映射算法          → 数据→音符
├── AI层
│   └── Kimi/OpenAI       → 趋势分析→作曲

    └── Three.js          → 3D音符粒子
```
